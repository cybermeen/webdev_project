WORKFLOW 2: Daily Planning and Task Management
Contributor: Aabeerah Iqbal

Purpose: 
This workflow allows users to organize their day by creating, viewing, editing, and completing tasks, helping them stay on top of their priorities and track their progress.

Steps:
1. User accesses their dashboard and views their current task list
2. User clicks "Add New Task" button to create a task
3. User enters task details (title, description, due date, priority level)
4. System saves the task and displays it in the appropriate section (e.g., "Today," "This Week," "Upcoming")
5. User can edit existing tasks by clicking on them and modifying details
6. User marks tasks as complete by checking the checkbox next to each task
7. Completed tasks move to a "Completed" section or are visually distinguished
8. User can delete tasks if needed using a delete button

End Result: 
The user has a clear, organized view of their daily tasks with the ability to track what's been accomplished. They maintain an up-to-date task list that helps them plan their day effectively and monitor their productivity progress.
